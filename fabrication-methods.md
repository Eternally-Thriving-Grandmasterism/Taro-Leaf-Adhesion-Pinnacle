# Taro Leaf Adhesion Fabrication Methods Lattice

Recommended Scalable Path: Multi-Level Nanoimprint Lithography (NIL) + Dense Wrinkle Texturing

1. Master Mold Preparation
   - Si/Ni shim: e-beam/LIL for 5–15 μm dense papillae + 50–300 nm wrinkle overlay

2. Resin Coating & Imprint
   - R2R UV resin on PET/PC → sequential NIL (micro papillae → nano wrinkles)

3. Demold & Surface Modification
   - Separate → optional plasma + hydrophobic coat (fluorosilane / wax) for pinning enhancement

4. Wrinkle Induction
   - Strain-release or thermal shrinkage for nanoscale folds on pillars

Alternative Paths:
- Colloidal + plasma wrinkling (spheres for micro → buckling for nano)
- Femtosecond laser + chemical texturing (direct on polymer/metal)

Valence Check:
- Dry/low-waste processes preferred
- Recyclable masters
- Passive function → eternal mercy
